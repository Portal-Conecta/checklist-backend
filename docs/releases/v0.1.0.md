# Release v0.1.0

Data: 2026-06-01

## Resumo

Esta release consolida a primeira versao funcional da Checklist API ao fim da sprint. O foco foi estruturar o modulo, implementar os principais fluxos de checklist, preparar a autenticacao com token do Hub e deixar os providers prontos para evoluir da camada mockada para a integracao real.

## Entregas principais

- Criacao da arquitetura base da Checklist API.
- Implementacao dos fluxos de template de checklist.
- Implementacao dos fluxos de rascunho, envio e cancelamento de checklist.
- Autenticacao stateless com JWT emitido pelo Hub.
- Autorizacao baseada em tipo de usuario e papel na turma.
- Providers de usuario, turma e sala com suporte a mock e HTTP.
- Bloqueio de perfis gerenciais em rotas operacionais de checklist.
- Regras de negocio para envio, itens obrigatorios e duplicidade.
- Tratamento global de erros.
- Documentacao de apoio para testes com Postman e autenticacao mockada.
- JavaDocs para facilitar leitura e manutencao do codigo.
